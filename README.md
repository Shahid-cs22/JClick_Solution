## 🧠 Introduction

The **MERN stack** is a popular JavaScript tech stack used to build **full-stack web applications**.

- **M** - MongoDB (Database)
- **E** - Express.js (Backend Framework)
- **R** - React.js (Frontend Library)
- **N** - Node.js (Runtime Environment)

---
